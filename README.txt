# منصة أم البنين التعليمية - دليل المشروع

## نظرة عامة
منصة تعليمية شاملة لرياض الأطفال والمدارس الابتدائية، مصممة لتسهيل التعلم والتواصل بين الطلاب، المعلمين، أولياء الأمور، والإدارة. المنصة تدعم التعلم التفاعلي، المتابعة اليومية، والإدارة الفعالة.

## الهوية البصرية
- **الألوان**: أساسي (#2E7D32)، ثانوي (#FBC02D)، خلفية (#F5F5F5 / #121212 للوضع المظلم)، نص (#212121).
- **الخطوط**: Cairo / Tajawal للعربية، Poppins / Roboto للإنجليزية.
- **الأيقونات**: Material Icons / Font Awesome مع تخصيص كرتوني للطلاب.

## البوابات الرئيسية

### 1. بوابة الإدارة
- إدارة المستخدمين (إضافة/تعديل/حذف، بحث).
- إدارة المناهج (إضافة دروس، نشر، تنظيم).
- التقارير (رسوم بيانية، تصدير PDF).
- الإعدادات (لغة، إشعارات، تخصيص هوية).
- الرسوم والمالية (إدارة مدفوعات أولياء الأمور، إيصالات، تقارير مالية).

### 2. بوابة المعلمين
- إنشاء درس (محتوى، فيديو، صور).
- متابعة الطلاب (حضور، أداء، تفاعل).
- إدارة الواجبات (إضافة، تصحيح، متابعة).
- التواصل (رسائل، اجتماعات مع ولي الأمر).
- تقارير التقدم.

### 3. بوابة أولياء الأمور
- متابعة الأداء (رسوم بيانية، تقارير أسبوعية).
- الجدول الدراسي (مواعيد الدروس والأنشطة).
- التواصل مع المعلمين (محادثة، طلب اجتماع).
- متابعة الرسوم والدفع الإلكتروني.
- الاطلاع على الأخبار والإعلانات.

### 4. بوابة الطلاب
- الدروس (قائمة دروس، فيديوهات، مواد).
- الواجبات (مهام جديدة، حالة التسليم).
- الأنشطة (ألعاب تعليمية، أنشطة تفاعلية).
- الجوائز (نظام نقاط، إنجازات، مكافآت).

## التدفقات (Flows)
- تسجيل الدخول: إدارة (بريد وكلمة مرور)، معلم/ولي أمر (OTP برقم الهاتف).
- Offline Mode: تخزين محلي + مزامنة عند الاتصال.
- أمان: Firebase Auth، تشفير بيانات، صلاحيات حسب الدور.

## تعليمات التنفيذ
- التصميم عبر Figma.
- التطوير: React Native + مكتبات (gesture-handler, reanimated, tts).
- الاختبارات: مع المعلمين، الطلاب، أولياء الأمور لضمان سهولة الاستخدام.

## حالة المشروع الحالية
- تم إنشاء مشروع Expo جديد.
- تم إعداد بوابة الإدارة وبوابة أولياء الأمور الأساسية.
- التنقل بين البوابات جاهز.
- يحتاج إلى توسيع الوظائف وإضافة بوابات أخرى.
